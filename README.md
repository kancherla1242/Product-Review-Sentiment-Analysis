# Product-Review-Sentiment-Analysis
Develop an NLP-based sentiment analysis model that automatically classifies Amazon product reviews as positive or negative, enabling efficient analysis of customer feedback and supporting data-driven business decisions.
#  Product Review Sentiment Analysis

## Overview

This project uses Natural Language Processing (NLP) and Machine Learning to classify Amazon product reviews as Positive or Negative.

## Dataset

Amazon Product Reviews Dataset from Kaggle.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TF-IDF Vectorization
* Logistic Regression
* Matplotlib
* Seaborn
* WordCloud

## Project Workflow

1. Data Cleaning and Preprocessing
2. Sentiment Label Generation
3. Text Vectorization using TF-IDF
4. Model Training using Logistic Regression
5. Performance Evaluation
6. Visualization using Word Cloud and Confusion Matrix

## Results

* Model: Logistic Regression
* Feature Extraction: TF-IDF
* Accuracy: 92.59%

## Visualizations

* Sentiment Distribution
* Positive Review Word Cloud
* Confusion Matrix

## Future Improvements

* BERT-based sentiment analysis
* Deep Learning models (LSTM/GRU)
